# Análise de Carteira de Investimentos em Python

Projeto educacional em Python para estudo de:
- Alocação de carteiras por perfil de risco
- Avaliação de carteiras em diferentes cenários macroeconômicos
- Simulação de investimento em CDB atrelado ao CDI

## Estrutura do projeto

- `dados.py` → Dados fixos (categorias, cenários e notas)
- `carteira.py` → Funções de alocação, análise e visualização da carteira
- `cdb.py` → Simulação de investimento em CDB
- `main.py` → Script principal de execução

## Objetivo

Projeto desenvolvido com fins de aprendizado em Python, modularização de código e conceitos básicos de investimentos.

ATENÇÃO: Não se trata de recomendação financeira.
